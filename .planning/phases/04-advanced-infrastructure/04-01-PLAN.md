After the `full_history.append(step_info)` block (line ~131), before the loop continues, add callback invocation:

if self._callbacks:
    cb_info = {
        'iteration': self._iteration_offset + i,
        'best_value': self._best_value,
        'best_solution': self._best_solution,
        'n_evaluations': self._n_evaluations,
        'wall_time': time.perf_counter() - t0,
    }
    for cb in self._callbacks:
        if cb(cb_info) is True:
            wall_time = time.perf_counter() - t0

                n_evaluations=self._n_evaluations,
                wall_time=wall_time,

**Step 4: Update `cyopt/optimizers/differential_evolution.py` `run()` method** (per D-03)

Modify the `callback` inner function to invoke user callbacks:
```python
def callback(xk: np.ndarray, convergence: float = 0) -> bool:
    history.append(self._best_value)
    if self._callbacks:
        cb_info = {
            'iteration': len(history) - 1 + self._iteration_offset,
            'best_value': self._best_value,
            'best_solution': self._best_solution,
            'n_evaluations': self._n_evaluations,
            'wall_time': time.perf_counter() - t0,
        }
        for cb in self._callbacks:
            if cb(cb_info) is True:
                return True  # SciPy stops when callback returns True
    return False
```
