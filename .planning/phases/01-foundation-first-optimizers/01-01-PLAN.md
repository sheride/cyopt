Create cyopt/base.py with `class DiscreteOptimizer(ABC)`:

Constructor parameters (per D-01, D-02, D-03, D-04):
- fitness_fn: Callable[[DNA], float] -- the objective to minimize (D-02)
- bounds: Bounds -- tuple of (lo_inclusive, hi_inclusive) pairs
- seed: int | None = None
- cache_size: int | None = None
- record_history: bool = False (D-04)
- progress: bool = False

Internal state:
- self._fitness_fn, self._bounds stored
- self._rng = np.random.default_rng(seed) (CORE-05)
- self._cache = EvaluationCache(maxsize=cache_size) (CORE-03)
- self._best_solution: DNA | None = None
- self._best_value: float = float('inf') (minimization, D-02) (CORE-04)
- self._n_evaluations: int = 0

Methods:
- `_evaluate(self, dna: DNA) -> float`: Check cache first; if miss, increment n_evaluations, call fitness_fn, store in cache. Always convert dna to tuple(int(x) for x in dna) before lookup (Pitfall 4). Update best-so-far if value < self._best_value.
- `_random_dna(self) -> DNA`: Generate random tuple within bounds using self._rng.integers(lo, hi + 1) for each dimension (Pitfall 1: hi+1 for inclusive upper bound).
- `run(self, n_iterations: int) -> Result`: Main loop with tqdm wrapping (CORE-07), calls self._step(i) each iteration, appends best_value to history, appends step_info to full_history if record_history=True. Returns Result(best_solution, best_value, history, full_history, n_evaluations, wall_time). Uses time.perf_counter for wall_time.
- `@abstractmethod _step(self, iteration: int) -> dict | None`: Subclasses implement. Returns dict for full_history or None.
