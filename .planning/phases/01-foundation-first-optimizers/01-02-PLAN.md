Define module-level function:
```python
def hamming_neighbors(dna: DNA, bounds: Bounds) -> list[DNA]:
    """Generate all Hamming-distance-1 neighbors within bounds."""
    neighbors = []
    for i, (lo, hi) in enumerate(bounds):
        for val in range(lo, hi + 1):
            if val != dna[i]:
                neighbor = list(dna)
                neighbor[i] = val
                neighbors.append(tuple(neighbor))
    return neighbors
```

```python
NeighborFunction = Callable[[DNA, Bounds], list[DNA]]

class GreedyWalk(DiscreteOptimizer):
    def __init__(self, fitness_fn, bounds, *,
                 neighbor_fn: NeighborFunction | None = None,
                 seed=None, cache_size=None, record_history=False, progress=False):
        super().__init__(fitness_fn, bounds, seed=seed, cache_size=cache_size,
                         record_history=record_history, progress=progress)
        self._neighbor_fn = neighbor_fn or hamming_neighbors
```

Override `run()` to reset walker state before calling super().run():
- self._current: DNA | None = None (will be initialized to random point on first _step)

`_step(self, iteration: int) -> dict | None`:
- If self._current is None: initialize with self._random_dna(), evaluate it
- Generate neighbors via self._neighbor_fn(self._current, self._bounds)
- Evaluate all neighbors
- Find best neighbor (lowest value)
- If best_neighbor_value < current_value: move to best neighbor (self._current = best_neighbor)
- Else: stay (local minimum found for this step); optionally restart from random point
  - Per old code pattern: when stuck, generate new random starting point
- Return dict with 'position', 'value', 'moved' keys if record_history else None
