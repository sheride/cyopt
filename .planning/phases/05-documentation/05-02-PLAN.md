<interfaces>
cyopt public API (__init__.py exports):
```python
from cyopt import (
    GA, RandomSample, GreedyWalk, BestFirstSearch,
    BasinHopping, DifferentialEvolution, MCMC, SimulatedAnnealing,
    DiscreteOptimizer, Result, DNA, Bounds, FitnessFunction,
    Callback, CallbackInfo, CheckpointCallback,
)
```

Result dataclass fields:
- best_solution: DNA (tuple[int, ...])
- best_value: float
- history: list[float] (best-so-far at each iteration)
- full_history: list[dict] | None
- n_evaluations: int
- wall_time: float

Optimizer constructor pattern:
```python
optimizer = GA(fitness_fn, bounds, seed=42, progress=True)
result = optimizer.run(n_iterations=100)
result2 = optimizer.continue_run(50)  # continue from where left off
```

CheckpointCallback:
```python
cb = CheckpointCallback(path="checkpoint.pkl", every_n=10)
result = optimizer.run(100, callbacks=[cb])
# Resume:
optimizer2 = GA.load_checkpoint("checkpoint.pkl")
result2 = optimizer2.continue_run(50)
```

**Section 7: Callbacks and Early Stopping**
- Markdown cell: Explain callback system.
- Code cell: Define a simple logging callback and a stopping callback:
  ```python
  def log_callback(info):
      if info["iteration"] % 50 == 0:
          print(f"Iter {info['iteration']}: best = {info['best_value']:.1f}")
      return None

  def stop_at_zero(info):
      if info["best_value"] == 0:
          print(f"Optimal found at iteration {info['iteration']}!")
          return True
      return None

  ga2 = GA(sphere, bounds, seed=42)
  result2 = ga2.run(500, callbacks=[log_callback, stop_at_zero])
  print(f"Stopped after {len(result2.history)} iterations, best = {result2.best_value}")
  ```

  ```python
  import tempfile, os

  with tempfile.TemporaryDirectory() as tmpdir:
      ckpt_path = os.path.join(tmpdir, "checkpoint.pkl")

      cb = CheckpointCallback(path=ckpt_path, every_n=50)
      ga3 = GA(sphere, bounds, seed=42)
      result3 = ga3.run(100, callbacks=[cb])
      print(f"After 100 iters: best = {result3.best_value}")

      # Resume from checkpoint
      ga4 = GA.load_checkpoint(ckpt_path)
      result4 = ga4.continue_run(100)
      print(f"After 100 more: best = {result4.best_value}")
  ```
