cyopt.frst public API:
```python
from cyopt.frst import FRSTOptimizer, FRSTResult, frst_optimizer, patch_polytope

# patch_polytope() is called on import of cyopt.frst
# After import, CYTools Polytope objects gain:
#   polytope.prep_for_optimizers()  -> precomputes 2-face triangulation data
#   polytope.optimizer_bounds       -> Bounds tuple for DNA space
#   polytope.optimizer_data         -> dict with precomputed data

# frst_optimizer factory:
frst_opt = frst_optimizer(
    optimizer_class=GA,        # any DiscreteOptimizer subclass
    polytope=polytope,         # CYTools Polytope (after prep_for_optimizers)
    target_func=target_fn,     # callable(Triangulation) -> float
    seed=42,
    **optimizer_kwargs,
)
result = frst_opt.run(100)  # returns FRSTResult

# FRSTResult extends Result with:
#   result.best_triangulation   -> CYTools Triangulation
#   result.best_cy              -> CYTools CalabiYau (if target used CY)
```
