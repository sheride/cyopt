  **cyopt/optimizers/basin_hopping.py:**

  Import `hamming_neighbors` from `cyopt.optimizers.greedy_walk`. Import or define `random_single_flip` -- try importing from `cyopt.optimizers._neighbors`, and if that module doesn't exist yet (parallel wave), define it inline with the same signature.

  Define a module-level helper:
  ```python
  def _greedy_descent(dna: DNA, bounds: Bounds, evaluate_fn: Callable[[DNA], float]) -> DNA:
  ```
  From starting point, repeatedly: get `hamming_neighbors`, evaluate each, move to best if it improves, else return current. This is a simple steepest-descent loop that terminates at a local minimum. Max iterations = 100 to prevent infinite loops on flat landscapes.

  Class `BasinHopping(DiscreteOptimizer)`:
  - Constructor params: `fitness_fn, bounds, *, temperature=1.0, n_perturbations=1, local_minimize_fn=None, perturb_fn=None, seed, cache_size, record_history, progress`
  - Validate `temperature > 0`
  - `self._temperature = temperature`
  - `self._n_perturbations = n_perturbations` (number of random flips in perturbation; if using default perturb_fn, call random_single_flip n_perturbations times)
  - `self._local_minimize_fn = local_minimize_fn or _greedy_descent`
  - `self._perturb_fn = perturb_fn` (store raw; if None, use random_single_flip in _step)
  - `self._current: DNA | None = None`, `self._current_value: float = float("inf")`

  `_step()`:
  1. Lazy init: `_current = _random_dna()`, evaluate, then local minimize from that point, evaluate the local minimum
  2. Perturb: if custom perturb_fn, call it once; else apply random_single_flip `n_perturbations` times in sequence
  3. Local minimize from perturbed point: `local_min = self._local_minimize_fn(perturbed, self._bounds, self._evaluate)`
  4. `local_min_value = self._evaluate(local_min)`
  5. Metropolis: `delta = local_min_value - self._current_value`; accept if `delta < 0` or `rng.random() < np.exp(-delta / self._temperature)`
  6. Return dict with current, local_min, accepted, temperature if recording

  **cyopt/optimizers/differential_evolution.py:**
  ```python
  from scipy.optimize import differential_evolution
  ```

  Class `DifferentialEvolution(DiscreteOptimizer)`:
  - Constructor params: `fitness_fn, bounds, *, popsize=15, mutation=(0.5, 1), recombination=0.7, strategy="best1bin", seed, cache_size, record_history, progress`
  - Store all as `self._popsize`, `self._mutation`, `self._recombination`, `self._strategy`

  `_step()`: Raise `NotImplementedError("DifferentialEvolution delegates to SciPy and cannot be stepped individually. Use run() directly.")`

  `run(n_iterations)`:
  1. `history: list[float] = []`
  2. Define `wrapped(x)`: convert x to `tuple(int(xi) for xi in x)`, call `self._evaluate(dna)`, return value
  3. Define `callback(xk, convergence=0)`: append `self._best_value` to history, return False
  4. Transform bounds: `de_bounds = [(lo, hi + 1) for lo, hi in self._bounds]` (CRITICAL: half-open for integrality)
  5. `integrality = [True] * len(self._bounds)`
  6. `t0 = time.perf_counter()`
  7. Call `differential_evolution(wrapped, de_bounds, integrality=integrality, maxiter=n_iterations, rng=self._rng, popsize=self._popsize, mutation=self._mutation, recombination=self._recombination, strategy=self._strategy, callback=callback, tol=0, polish=False)`
  8. `wall_time = time.perf_counter() - t0`
  9. Return `Result(best_solution=self._best_solution, best_value=self._best_value, history=history, full_history=None, n_evaluations=self._n_evaluations, wall_time=wall_time)`

  Note: `record_history` and `progress` are accepted by constructor (via super().__init__) but DE does not produce full_history (always None) and does not use tqdm (SciPy manages its own loop). This is acceptable and documented in docstring.

  Both classes: full NumPy-style docstrings, type hints throughout. Zero CYTools imports.
