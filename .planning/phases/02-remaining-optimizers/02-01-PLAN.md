  **cyopt/optimizers/best_first_search.py:**
  Import `hamming_neighbors`, `NeighborFunction` from `cyopt.optimizers.greedy_walk`. Import `heapq`.

  Class `BestFirstSearch(DiscreteOptimizer)`:
  - Constructor params: `fitness_fn, bounds, *, mode="backtrack", neighbor_fn=None, seed, cache_size, record_history, progress`
  - Validate `mode in ("backtrack", "frontier")`, raise ValueError otherwise
  - Store `self._neighbor_fn = neighbor_fn or hamming_neighbors`
  - Store `self._mode = mode`
  - Initialize `self._current: DNA | None = None`, `self._current_value: float = float("inf")`
  - If backtrack: `self._path: list[DNA] = []`, `self._avoid: set[DNA] = set()`
  - If frontier: `self._frontier: list[tuple[float, int, DNA]] = []` (value, tiebreaker counter, dna), `self._visited: set[DNA] = set()`, `self._counter: int = 0`

  `_step()` dispatches to `_step_backtrack()` or `_step_frontier()`.

  `_step_backtrack()` (per user decision and research):
  1. Lazy init: if `_current is None`, set `_current = _random_dna()`, evaluate, push to `_path`
  2. Get neighbors via `self._neighbor_fn(self._current, self._bounds)`
  3. Filter: remove any in `self._avoid` and any in `self._path` (to avoid revisiting path nodes)
  4. If no valid neighbors: random restart (clear path, clear avoid, new random point)
  5. Evaluate each valid neighbor one at a time. If any improves (value < _current_value): move there immediately (relaxed greedy -- first improving, not best overall). Push old current onto path.
  6. If none improve after checking all: move to best neighbor anyway. Push old current onto path.
  7. Oscillation check: if `len(self._path) >= 2 and self._current == self._path[-2]`: add `self._path[-1]` to `self._avoid`, pop last 2 from path.

  `_step_frontier()`:
  1. Lazy init: if `_current is None`, set `_current = _random_dna()`, evaluate, mark visited
  2. Get neighbors, for each unseen: evaluate, add `(value, counter++, neighbor)` to heap, mark visited
  3. Pop best from heap (if empty: random restart). Set as current.

  **cyopt/optimizers/mcmc.py:**
  Import `random_single_flip`, `StepFunction` from `cyopt.optimizers._neighbors`.

  Class `MCMC(DiscreteOptimizer)`:
  - Constructor params: `fitness_fn, bounds, *, temperature=1.0, step_fn=None, seed, cache_size, record_history, progress`
  - Validate `temperature > 0`, raise ValueError otherwise
  - Store `self._temperature = temperature`, `self._step_fn = step_fn or random_single_flip`
  - Store `self._current: DNA | None = None`, `self._current_value: float = float("inf")`

  `_step()`:
  1. Lazy init _current
  2. `proposal = self._step_fn(self._current, self._bounds, self._rng)`
  3. `proposal_value = self._evaluate(proposal)`
  4. `delta = proposal_value - self._current_value`
  5. Accept if `delta < 0` or `self._rng.random() < np.exp(-delta / self._temperature)`
  6. Return dict with position, value, accepted bool if recording

  **cyopt/optimizers/simulated_annealing.py:**
  Import `random_single_flip`, `StepFunction` from `cyopt.optimizers._neighbors`.

  Class `SimulatedAnnealing(DiscreteOptimizer)`:
  - Constructor params: `fitness_fn, bounds, *, n_iterations=1000, t_max=1.0, t_min=1e-8, step_fn=None, seed, cache_size, record_history, progress`
  - Validate `t_max > t_min > 0` and `n_iterations > 0`
  - Store `self._n_iterations = n_iterations`, `self._t_max`, `self._t_min`, `self._step_fn = step_fn or random_single_flip`
  - Store `self._current: DNA | None = None`, `self._current_value: float = float("inf")`
  - Store `self._step_count: int = 0` (accumulated across run() calls for cooling schedule continuity)

  `_step()`:
  1. Lazy init _current
  2. Compute `frac = min(1.0, self._step_count / max(1, self._n_iterations - 1))`
  3. `temperature = self._t_max * (self._t_min / self._t_max) ** frac`
  4. Propose via `self._step_fn`, evaluate
  5. Metropolis acceptance with `temperature`
  6. Increment `self._step_count`
  7. Return dict with position, value, temperature, accepted if recording

  All classes: full NumPy-style docstrings on class and public methods. Type hints on all parameters and returns.
