<objective>
Implement the DNA encoding/decoding layer that maps between integer-tuple DNA and CYTools FRST triangulations, plus the Polytope monkey-patch that precomputes 2-face triangulation data.

Purpose: This is the core data bridge between the generic discrete optimizers (Phase 1-2) and CYTools' FRST pipeline. Without it, the wrapper layer (Plan 02) has nothing to encode/decode.

Output: `cyopt/frst/_encoding.py` with all encoding functions, `cyopt/frst/__init__.py` with import guard and auto-patch, comprehensive unit tests proving roundtrip correctness.

**`cyopt/frst/_encoding.py`:** Contains all encoding logic per D-01, D-07, D-08, D-09.

Functions to implement:

1. `_normalize_simplices(simplices) -> frozenset[tuple[int, ...]]`: Normalize simplices (from np.ndarray or list[list[int]]) to frozenset of sorted int tuples for O(1) comparison.

2. `_prep_for_optimizers(self, **kwargs) -> None`: Monkey-patched method on Polytope.
   - Guard: if `getattr(self, '_cyopt_prepped', False)` return immediately (idempotent).
   - Guard: if not `self.is_reflexive()` raise `ValueError("FRST optimization requires reflexive polytopes.")`.
   - Call `self.face_triangs(**kwargs)` and store as `self._cyopt_face_triangs`.
   - Identify interesting faces (indices where `len(face_ts) > 1`) and store indices in `self._cyopt_interesting: list[int]`.
   - Compute bounds as `tuple((0, len(face_ts) - 1) for i, face_ts in enumerate(self._cyopt_face_triangs) if len(face_ts) > 1)` and store as `self._cyopt_bounds: Bounds`.
   - Precompute simplex sets for reverse mapping: for each face's triangulations, store `_normalize_simplices(ft.simplices())` per triangulation. Store as `self._cyopt_face_simp_sets: list[list[frozenset]]`.
   - Set `self._cyopt_prepped = True`.

3. `_dna_to_frst(self, dna: DNA) -> object | None`: Monkey-patched method.
   - Build a list of `None` entries (length = total number of faces).
   - For each index in `self._cyopt_interesting`, set the corresponding face's Triangulation from `self._cyopt_face_triangs[face_idx][dna[i]]`.
   - Call `self.triangfaces_to_frst(triangs)` (per D-08) and return the result (Triangulation or None for non-solid cones).

4. `_dna_to_cy(self, dna: DNA) -> object | None`: Monkey-patched method.
   - Call `triang = self.dna_to_frst(dna)`. If None, return None.
   - Return `triang.get_cy()`.

5. `_triang_to_dna(self, triang) -> DNA`: Monkey-patched method.
   - Call `restrictions = triang.restrict()` (restrict_dim=2 default).
   - For each interesting face index, normalize the restriction's simplices and match against precomputed `_cyopt_face_simp_sets[face_idx]`.
   - Return tuple of matched indices.
   - Raise `ValueError` if any face's restriction doesn't match any known triangulation.

6. `_cy_to_dna(self, cy) -> DNA`: Monkey-patched method.
   - Get the triangulation from the CY: `triang = cy.triangulation()` (or equivalent CYTools accessor).
   - Return `self.triang_to_dna(triang)`.

7. `patch_polytope() -> None`: Import `Polytope` from cytools. Attach all functions above as bound methods:
   - `Polytope.prep_for_optimizers = _prep_for_optimizers`
   - `Polytope.dna_to_frst = _dna_to_frst`
   - `Polytope.dna_to_cy = _dna_to_cy`
   - `Polytope.triang_to_dna = _triang_to_dna`
   - `Polytope.cy_to_dna = _cy_to_dna`
